Featured Project FPGA • VHDL

UART IP Core with Auto Baud Rate Detection

Context

Designed as a digital communication project for FPGA-based embedded systems requiring flexible UART reception without manual baud rate configuration.

Problem

Fixed baud rate configurations reduce interoperability and make serial communication less adaptable when devices operate under varying timing conditions.

Solution

Implemented a synthesizable UART IP Core in VHDL with dynamic baud rate detection based on synchronization timing from the 0x7F frame, including rising-edge timing analysis and synchronization for accurate bit sampling.

Technologies

VHDL FPGA UART ICEcube2 ModelSim iCE40HX1K

Results / Impact

Validated robust baud detection over standard UART rates, produced a compact reusable IP block, and reinforced practical skills in FPGA design flow, simulation, timing logic, and hardware-oriented verification.
