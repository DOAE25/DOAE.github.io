Featured Project FPGA • VHDL

UART IP Core with Auto Baud Rate Detection

Context

Designed as a digital communication project for FPGA-based embedded systems requiring adaptable serial interfaces and robust synchronization behavior.

Problem

Fixed baud rate configurations reduce interoperability and make serial communication less adaptable when devices operate under varying timing conditions.

Solution

Implemented a synthesizable UART IP Core in VHDL with dynamic baud rate detection based on synchronization timing from the 0x7F frame, including a dedicated synchronization module for accurate bit sampling and timing alignment.

Technologies

VHDL FPGA UART ICEcube2 ModelSim iCE40HX1K

Results / Impact

Validated robust baud detection over standard UART rates, produced a reusable communication block for embedded architectures, and strengthened practical skills in FPGA flow, timing logic, and verification for hardware-centered systems.
